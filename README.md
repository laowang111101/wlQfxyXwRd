# Python电影推荐系统 python562

## 项目概述

本项目是一个基于Django框架开发的Python电影推荐系统，设计于2025年暑期。系统使用最新技术，为用户提供个性化电影推荐，同时包含完整的管理端功能。

## 技术栈

- Python版本：3.9及以上（建议3.11）
- 开发框架：Django
- 开发工具：Pycharm
- 数据库：Mysql 8.0及以上
- 数据库软件：Navicat或Dbeaver

## 功能模块

### 用户端功能

- 首页
- 每日推荐
- TOP10
- 分类/标签
- 推荐系统（基于内容/热度/兴趣推荐）
- 影片详情
- 影片评分
- 数据统计（数据可视化）
- 个人信息管理
- 登录注册

### 管理端功能

- 影片管理
- 分类/标签管理
- 偏好管理
- 用户管理

## 系统角色

- 普通用户：使用推荐系统进行电影浏览与选择
- 管理员：负责影片、分类、用户等的管理和系统维护

## 运行环境

- 推荐使用Linux或MacOS操作系统
- Python 3.9及以上版本
- Mysql 8.0及以上版本
